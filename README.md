# Maple Cast Fishing Tackle

Static homepage for Maple Cast, prepared for GitHub backup and Vercel deployment.

## Structure

- `index.html` - deployable homepage
- `assets/` - logo, packaging, and fishing imagery
- `vercel.json` - Vercel static-site settings

## Deploy

Import this GitHub repository into Vercel. Vercel can deploy it as a static project with no build command and `.` as the output directory.
