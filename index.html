Which lure should I start with?

Start with jig and vib. Jig is a reliable first choice for control and bottom contact; vib is great when you want flash, vibration, and faster searching.

Yes. Accessories can include terminal tackle, boxes, hooks, snaps, and other support gear for lure fishing.
